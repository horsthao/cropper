About butterfly cropping

Butterflies are an interesting species. Knowing about how populations of butterflies change can help scientists answer questions about climate change, evolution and biodiversity.

At UCL we are working on automatic tools which can be useful for ecologists, and for these tools we need data. Cropped images of butterflies are one important source of data for the tools we are building.

How do you stop cheaters?

We realise that someone who wanted to be top of the leaderboard could cheat, for example by drawing a random rectangle instead of taking time to draw a good crop.

We take two steps to prevent this:

  • Each image is given to multiple users. These multiple labels can be used to eliminate outliers
  • Every so often, we will serve an image to a user for which we have carefully labelled a crop box. Users who consistently give wildly different crops to our 'gold standard' are not to be trusted.

What if I make a mistake?

Don't worry if you realise you've made a mistake on an image! We serve each image to multiple users so mistakes will get eliminated in the long run.

In future, we may introduce an 'undo' button to allow users to correct submitted mistakes

Technical bits

  • The back-end is written in Python, using Flask and Jinja
  • The leaderboard uses Chartkick
  • We use the standard Boostrap theme
  • The cropping interface uses cropper.js
